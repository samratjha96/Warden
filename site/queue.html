How the Queue Works
01 Submit a GitHub/GitLab repository URL on the home page.
02 The job is added to data/queue/index.json with status "pending".
03 An async worker (outside this static site) picks up pending jobs.
04 The LLM-powered analyzer generates a structured JSON report.
05 Report is saved to data/reports/{id}.json and index updated.
06 This static site renders the report from the JSON file.
