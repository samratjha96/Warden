06 — Code Safety · HIGH
HIGH eval() / new Function() on attacker-reachable input

Browser interaction tooling constructs and evaluates function bodies dynamically. The intended use is browser-context automation, but if attacker-controlled input reaches these code paths, arbitrary code execution is the result — with no sandbox.

09 — Supply Chain · HIGH
HIGH Remote installer: curl … | bash

The macOS app installer builds and executes a remote shell script via curl | bash. This gives the install process full host execution privileges from an unverified remote source — MitM or CDN compromise leads directly to RCE. Same pattern appears in docker smoke/e2e test scripts.

10 — Permissions · HIGH
HIGH Chrome extension: debugger + tabs — near-total browser control

The extension requests debugger, tabs, activeTab, storage, alarms, and webNavigation. Host permissions are loopback-only, but the debugger + tabs combination enables read/write access to all browser tabs, including credential fields and authenticated sessions.

Android manifest requests five categories of sensitive permissions. Cleartext traffic is also permitted in defined contexts. The combination makes this app unsuitable for deployment on managed corporate devices.

15 — Remediation (if evaluation required)

APPROVE WITH CONDITIONS only under strict controls:

01
Run only in an isolated VM or container with no privileged corp credentials or host mounts.
02
Enforce outbound network allowlists — permit only provider domains strictly required for your use case.
03
Disable or remove the OTel diagnostics plugin unless explicitly needed.
04
Disable remote installer one-liners. Use pinned artifact + checksum/SBOM verification instead of curl | bash.
05
Restrict exec tooling by policy — strict allowlists and explicit approvals before any shell execution.
06
Verify gradle-wrapper.jar provenance in CI via Gradle wrapper validation and checksum policy.
07
Prefer tagged releases and pinned commits. Require branch protection and signed-release verification for internal adoption.
